# 🧬 API-DNA Chrome Extension

Instant API verification in your browser. Detect fake models, token inflation, and relay servers with one click.

## Features

- 🔍 **One-click API scan** — Verify any API endpoint instantly
- ⚠️ **Fake model detection** — GPT-3.5 sold as GPT-4? We catch it
- 💸 **Token inflation alerts** — Being overcharged? We prove it
- 🔀 **Relay detection** — Proxy servers intercepting your calls? We find them

## Install

1. Clone this repo
2. Open `chrome://extensions/` in Chrome
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder
